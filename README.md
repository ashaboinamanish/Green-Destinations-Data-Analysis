# Employee Attrition Analysis - Green Destinations 📉🏢

This project analyzes employee attrition data from Green Destinations, a well-known travel agency, to identify patterns and factors influencing employees to leave the organization.

## Objective
The HR Director noticed a rise in employee attrition and wants to:
- Calculate the overall **attrition rate** (% of employees who have left).
- Identify whether factors like **age**, **income**, and **years at the company** impact attrition.

## Contents
- `green_destinations_attrition.ipynb` – Main analysis notebook
- Data cleaning and preprocessing steps
- Exploratory Data Analysis (EDA)
- Attrition rate calculations
- Visualizations of key factors affecting attrition
- Summary of findings and recommendations

## Tools Used
- Python 🐍
- Pandas 🧮
- Matplotlib & Seaborn 📊
- Jupyter Notebook 📓
